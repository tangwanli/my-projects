餐厅介绍

Company Profile

  尚CLUB餐厅位于英国伦敦唐人街繁华地段，地理位置优越，占地面积200平方米，楼高两层，餐厅环境优雅、宽敞明亮。 餐厅以“人无我有、人有我精、取长补短、自立门户”的理念，潜心研究，创制出兼融南北菜系所长，更适合当今人口味的新派菜肴，北方口味为主，兼顾各地不同的饮食习惯和口味特点，吃起来既符合大众人的胃口，同时也能够满足来自不同地域人们的不同口味需求。 餐厅可以延开10-15席，可以容纳100-150人用餐，是接待中国团队中餐、商务宴请的首选。 出品至上，服务跟上，是尚CLUB中餐厅一直强调的作法。为了提升服务素质，我们每天进行餐前培训，餐后总结，根据团队种类不同提出了个性化服务、亲情化服务，让游客满意，真正理解顾客就是我们服务的真缔。
